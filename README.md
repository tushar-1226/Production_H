# LIQUID
Liquid is a full-stack e-commerce web application where users can explore, search, and purchase all types of drinks — from soft drinks and juices to energy drinks and more — all in one platform.

## Feature
Browse a wide variety of drinks in one place
Search and filter drinks
User authentication (Signup/Login)
Add to cart & manage orders

## TechStack
Frontend: React
Backend: Node.js + Express
Database: MongoDB
Authentication: JWT
Other Tools: Git, Postman
